Verser un exercice (latex)

Si le fichier latex était le suivant, il devrait pouvoir se compiler par pdflatex : 

\documentclass[12pt,frenchb]{article}
\usepackage{amsmath,amssymb,babel,multicol,enumerate,graphicx,tabularx,textcomp}
\usepackage{pgf,tikz}
\usetikzlibrary{shapes,positioning,arrows,patterns}
\usepackage[T1]{fontenc}
\usepackage[utf8]{inputenc}

\newcommand{\ds}{\displaystyle}
\newcommand{\C}{{\mathbb C}}
\newcommand{\R}{{\mathbb R}}
\newcommand{\Q}{{\mathbb Q}}
\newcommand{\Z}{{\mathbb Z}}
\newcommand{\N}{{\mathbb N}}

\begin{document}
    \begin{enumerate}
	\item
        {

Texte latex à verser sur le serveur 

        }
    \end{enumerate}
\end{document}


Remarques :

  • Vous pouvez définir des commandes personnelles par \newcommand au début de votre texte. Comme celui-ci se trouve entre { et }, ces commandes restent restreintes à votre exercice.
  • Si vous voulez absolument utiliser pstricks (au lieu de tikz) pour vos graphiques, suivez ce lien pour la démarche compatible avec le serveur.
